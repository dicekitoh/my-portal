Claude Code スキル一覧

2026年2月22日 現在

11 skills

Claude Code で利用可能なスキル（スラッシュコマンド）の一覧です。各スキルはキーワードまたは /コマンド名 で呼び出せます。

デバイス・接続

デバイス一括接続 /connect
自宅3台のPC（Linux / macOS / Windows）にSSH接続して状態を一覧表示する。
発動: すべてのデバイスに接続, 自宅に接続, リモート接続, デバイス確認

情報検索・AI

マルチAI相談 /ask-ai
複数AI（Claude・Gemini・Grok・Perplexity・Codex）に同じ質問をして回答を比較する。
発動: 3つのAIに聞いて, Geminiの意見も聞きたい, マルチAI相談
X リアルタイム検索 /x-search
X（旧Twitter）をGrok API経由でリアルタイム検索する。
発動: X検索, Xで調べて, ツイート検索, Xで何か話題

ブログ・コンテンツ

ブログ投稿 /blog
Blogger APIで5つのブログに記事を執筆・投稿する。
発動: ブログ投稿, ブログに書いて, 記事投稿, ブログ下書き

タスク・スケジュール

TODO三方向同期 /todo
URLからTODOを自動生成し、Google Tasks / Apple Reminders / Microsoft To Do に追加・同期する。
発動: TODO作成, これTODOにして, 同期チェック
HOTEL FORESTY 登録 /hotel
HOTEL FORESTY の予定をGoogle Calendarに登録・更新する。
発動: ホテル登録, FORESTY, /hotel 2/16 10:00-15:30
勤務準備 /shift-prep
勤務前の準備を対話形式で組み立て、TODOとカレンダー予定を自動作成する。
発動: 勤務準備, シフト準備, /shift-prep 2/19

データ収集・更新

シティヘブンDB /ch
シティヘブン北海道DBの更新・検索・分析を行う。
発動: CH更新, シティヘブン, キャスト検索, キャスト分析
オービスデータ更新 /orbis
速度測定装置（オービス）のデータを差分更新・確認する。
発動: オービス更新, オービス確認, orbis log

設定

キーバインド設定 /keybindings-help
キーボードショートカットのカスタマイズ・リバインド設定。
発動: rebind ctrl+s, customize keybindings
